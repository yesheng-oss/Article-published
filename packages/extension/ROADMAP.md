# WechatSync v2.0 Roadmap

## 🚀 功能增强

### 发布能力
- [ ] 直接发布（非草稿）选项 — 逐平台适配，需研究各平台发布 API
  - 优先：CSDN（改 status）、知乎、掘金（有 publish 接口）
  - 中等：头条、百家号、B站（需额外参数）
  - 不建议：公众号（不可撤回）、小红书（审核严格）
  - 注意：当前所有 adapter 的 `draftOnly` 参数仅透传，未实现分支逻辑
- [ ] 批量同步多篇文章

### 内容处理
- [ ] 多尺寸封面自动适配（公众号 2.35:1、小红书 3:4、知乎/头条 16:9）
  - 短期：canvas 自动裁剪
  - 长期：支持为不同平台组指定不同封面
- [ ] 自定义图床支持
- [x] 去除平台外链装饰（微信已完成）

### 用户体验
- [x] Popup 渐进式引导（空状态引导、平台登录提示、首次同步成功提示）
- [x] 微信编辑页 popup 同步可用
- [x] 编辑器可编辑性视觉提示
- [ ] 同步失败错误信息人性化（当前报错偏技术化）
- [ ] 平台 icon 本地化缓存（避免外链失效，如豆瓣 CDN 418）

### 工具链
- [x] MCP Server（WebSocket 桥接，Claude Code 集成）
- [x] CLI 工具（命令行同步）
- [ ] Firefox / Edge 版本
